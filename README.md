## 🌌 项目特色

- 📊 **多维度词云**：今日/昨日/本周/上周/本月/上月/年度词云
- 🎯 **词性词云**：名词/动词/形容词/副词，分类展示群聊话题
- 📈 **热词趋势**：对比分析新兴/衰退话题
- 🏆 **发言排名**：展示群聊活跃度排行
- 🔍 **词性分析**：统计词类分布，了解群聊语言风格
- 🎨 **语言画像**：群聊语言画像 + 个人风格分析
- 📅 **定时发送**：每日自动推送词云
- 🖼️ **自定义形状**：设置群级词云遮罩
- 📝 **群级词典**：添加专业术语，提高分词精度
- 🎨 **丰富配置**：颜色、字体、背景等高度可定制

### ⚔️ 用户对比

| 指令 | 别名 | 功能 |
|------|------|------|
| `比较词云 @用户A` | `对比 @用户A`、`比较 @用户A` | 将用户A与自己对比（默认本月） |
| `比较词云 @用户A @用户B` | `对比`、`比较` | 对比两位用户的发言风格（默认本月） |
| `比较词云 今天 @用户A @用户B` | `对比 今天`、`比较 今天` | 对比今日发言风格 |
| `比较词云 本周 @用户A @用户B` | `对比 本周`、`比较 本周` | 对比本周发言风格 |
| `比较词云 本月 @用户A @用户B` | `对比 本月`、`比较 本月` | 对比本月发言风格 |
| `比较词云 今年 @用户A @用户B` | `对比 今年`、`比较 今年` | 对比今年发言风格 |

**对比维度：**
- 📊 基础数据：发言数、平均消息长度、词汇丰富度、活跃时段
- 🏷️ 风格标签：共同标签 vs 差异标签
- 📝 词性对比：名词/动词/形容词/副词占比对比
- 🔤 高频词：共同高频词 vs 独特词汇
- 📈 相似度评分：综合评估两位用户的语言风格相似程度

## ⚙️ 配置说明

在 AstrBot 管理面板中可配置以下选项：

### 基础设置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `wordcloud_pkuseg_model` | 选项 | `web` | pkuseg 分词模型（web/news/medicine/tourism） |
| `wordcloud_min_word_length` | 数字 | `2` | 词云中显示的最小词语长度，小于此长度的词将被过滤 |
| `wordcloud_max_words` | 数字 | `200` | 词云图片中最多显示的词语数量 |
| `wordcloud_width` | 数字 | `800` | 生成词云图片的宽度（像素） |
| `wordcloud_height` | 数字 | `600` | 生成词云图片的高度（像素） |
| `wordcloud_background_color` | 选项 | `white` | 词云图片的背景颜色（white/black/transparent） |
| `wordcloud_colormap` | 选项 | `viridis` | 词云字体颜色方案 |
| `wordcloud_filter_function_words` | 开关 | `true` | 开启后过滤代词、介词、连词、助词等虚词，仅保留实词 |

### 文件上传

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `wordcloud_font_file` | 文件 | - | 上传自定义字体文件（.ttf/.otf），留空则使用内置默认字体 |
| `wordcloud_stopwords_file` | 文件 | - | 上传自定义停用词表（每行一个词），留空则使用内置默认停用词表 |
| `wordcloud_user_dict_file` | 文件 | - | 上传 pkuseg 用户词典，用于添加领域专有词汇（格式：词语 词性，每行一个） |

### 排名设置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `wordcloud_ranking_limit` | 数字 | `10` | 发言排名中显示的最大人数 |
| `wordcloud_ranking_show_percentage` | 开关 | `true` | 发言排名中是否显示每人消息数占总数的百分比 |

### 词性词云颜色

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `wordcloud_pos_noun_colormap` | 选项 | `Blues` | 名词词云的字体颜色方案 |
| `wordcloud_pos_verb_colormap` | 选项 | `Greens` | 动词词云的字体颜色方案 |
| `wordcloud_pos_adj_colormap` | 选项 | `Oranges` | 形容词词云的字体颜色方案 |
| `wordcloud_pos_adv_colormap` | 选项 | `Purples` | 副词词云的字体颜色方案 |

### 热词趋势

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `wordcloud_trend_threshold` | 数字 | `0.5` | 词频增长率超过此阈值（50%）视为新兴热词，低于负阈值视为衰退话题 |
| `wordcloud_trend_emerging_limit` | 数字 | `10` | 热词趋势报告中新兴热词的最大显示数量 |
| `wordcloud_trend_declining_limit` | 数字 | `5` | 热词趋势报告中衰退话题的最大显示数量 |

### 语言画像

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `wordcloud_profile_top_words` | 数字 | `5` | 群聊语言画像中高频词的显示数量 |
| `wordcloud_style_adj_threshold` | 数字 | `0.15` | 个人形容词占比超过此值（15%）标记为「感性表达者」 |
| `wordcloud_style_verb_threshold` | 数字 | `0.30` | 个人动词占比超过此值（30%）标记为「行动派」 |
| `wordcloud_style_noun_threshold` | 数字 | `0.40` | 个人名词占比超过此值（40%）标记为「知识分享者」 |

## 🛠️ 技术栈

- **分词引擎**：spacy-pkuseg（优先）/ jieba（降级兼容）
- **词云生成**：wordcloud
- **图像处理**：Pillow
- **矩阵运算**：numpy
- **异步请求**：aiohttp

## ⚠️ 注意事项

1. **前置依赖**：本插件依赖 `astrbot_plugin_message_recorder` 插件提供消息数据，请确保已安装并启用。
2. **分词引擎降级**：如果 pkuseg 初始化失败（如权限问题），插件会自动降级到 jieba 分词引擎，无需手动干预。
3. **字体加载**：插件内置 HarmonyOS Sans SC 字体，同时支持自定义字体上传。若中文字体不可用，词云将无法显示中文。
4. **群级配置隔离**：词云遮罩、自定义词典等配置按群隔离，不同群聊互不影响。
5. **定时任务**：定时发送功能基于插件内部轮询实现，Bot 重启后已保存的定时配置会自动恢复。
